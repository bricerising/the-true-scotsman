# Templates Index

- `app-repo/AGENTS.md`: agent instructions template for app repositories
- `ci/github-actions-quality.yml`: baseline CI quality gates (lint/typecheck/test/build/audit)
- `service-spec-bundle/`: starter `spec.md`, `plan.md`, `tasks.md`, `quickstart.md`, `data-model.md`, and contract guidance
